# **App Name**: Frango Assado Delivery

## Core Features:

- Menu Display: Exibir o cardápio de frango assado e acompanhamentos com fotos, descrições e preços.
- Shopping Cart Management: Permitir aos usuários adicionar, remover e ajustar a quantidade de itens no carrinho de compras antes de finalizar o pedido.
- Order Submission: Facilitar o processo de finalização do pedido, coletando informações de entrega e contato do cliente.
- Payment Processing: Integrar com métodos de pagamento para processar transações de forma segura e eficiente.
- Order History and Status Tracking: Permitir aos usuários visualizar o histórico de seus pedidos e acompanhar o status atual da entrega em tempo real.
- Admin Dashboard: Oferecer uma interface para administradores gerenciarem produtos do cardápio, processarem novos pedidos e monitorarem o status das entregas.
- Basic PWA Functionality: Configurar o aplicativo para funcionalidades PWA básicas, como instalação na tela inicial, caching de assets e acesso offline para melhor experiência do usuário.

## Style Guidelines:

- O fundo da página inicial apresenta um gradiente vermelho escuro, evocando calor e apetite.
- Cores primárias e secundárias inspiradas em tons terrosos e quentes, como marrom, laranja queimado e creme, para complementar o tema de comida assada.
- Cor de destaque para botões de ação e informações importantes: um tom de amarelo mostarda ou dourado para chamar a atenção e indicar interação.
- Fontes sans-serif amigáveis e legíveis para o corpo do texto, como 'Roboto' ou 'Inter', garantindo clareza nas descrições dos produtos.
- Para títulos e o nome do restaurante, uma fonte com um toque mais 'artesanal' ou 'premium' para transmitir qualidade e tradição.
- Ícones simples e reconhecíveis para navegação (home, cardápio, carrinho, perfil), bem como para indicar status de pedido e opções de pagamento (frango, carrinho de compras, relógio, etc.).
- Layout limpo e intuitivo, com foco na visualização clara dos produtos. Design responsivo, otimizado para dispositivos móveis e fácil navegação com uma mão.
- Página inicial com um título centralizado 'Frango Assado' em destaque e um botão proeminente 'Ver Cardápio' logo abaixo para direcionar o usuário.
- Animações sutis e rápidas para feedback visual ao adicionar itens ao carrinho, atualizar a quantidade ou transicionar entre páginas, melhorando a experiência sem distrair.
